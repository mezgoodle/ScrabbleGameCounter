# Scrabble Tracker

A minimalist, dark-mode Scrabble Scoreboard and Word Score Calculator for the Ukrainian version of the board game.

## Features:
- **Ukrainian Layout Support**: Incorporates the accurate letter values for the Ukrainian alphabet (including the apostrophe and specific values like С - 2, Щ - 8, Ґ - 10, etc.).
- **Score Calculator**: Automatically calculates points based on the entered word.
- **Word and Letter Multipliers**: Supports board bonuses for letters (x2, x3) and words (x2, x3), as well as the "Bingo" bonus (+50 points for using all 7 tiles).
- **Player Management**: Configure the game for 1 to 4 players with named profiles and real-time ranking tracking.
- **Turn History**: Logs the last 5 turns with the ability to undo or edit a turn if a mistake was made.
- **Match Statistics**: Displays the average score, total words played, and announces the final winner of the game.

## Technologies:
- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
